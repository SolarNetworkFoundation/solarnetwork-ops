SolarNetwork Foundation Debian Package Repository

This is the SolarNetwork Foundation Debian Package Repository.

Try adding an apt repo configuration /etc/apt/sources.list.d/solarnetwork.list with the following content:

deb https://debian.repo.solarnetwork.org.nz stretch main

You'll need to import the GPG signing key, which you can do via:

$ curl -O  https://debian.repo.solarnetwork.org.nz/KEY.gpg |sudo apt-key add -
