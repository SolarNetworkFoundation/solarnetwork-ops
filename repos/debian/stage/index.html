SolarNetwork Foundation Debian Package Repository (Staging)

This is the SolarNetwork Foundation Debian Package Repository (Staging).

Heads up! This repository is for testing and development and may contain broken or unstable packages. Unless you're quite sure this is what you need, use the production repository instead.

Try adding an apt repo configuration /etc/apt/sources.list.d/solarnetwork.list with the following content:

deb https://debian.repo.stage.solarnetwork.org.nz stretch main

You'll need to import the GPG signing key, which you can do via:

$ curl -s https://debian.repo.stage.solarnetwork.org.nz/KEY.gpg |sudo apt-key add -
